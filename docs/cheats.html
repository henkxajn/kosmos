KOSMOS — Cheaty konsoli

Komendy dostępne w konsoli przeglądarki (F12 → Console). Wszystkie pod window.KOSMOS.debug.
Jak użyć: otwórz grę w przeglądarce, naciśnij F12, przejdź do zakładki Console, wklej dowolną z komend poniżej i naciśnij Enter. Zmiany działają natychmiast, nie wymagają reloadu. Wszystko idzie przez normalne API gry (ResourceSystem, CivSystem), więc eventy i UI aktualizują się jak przy normalnej rozgrywce.

Zasoby i waluta

give(gains) — dowolne zasoby i commodities

// Dodaj konkretne ilości surowców lub towarów do aktywnej kolonii.
// Akceptuje surowce (Fe, C, Si, Cu, Ti, Li, Hv, Xe, Nt),
// utility (food, water, research) i commodities (structural_alloys, power_cells, ...).

KOSMOS.debug.give({ Fe: 1000, Ti: 500, Hv: 200 });

KOSMOS.debug.give({
  structural_alloys: 50,
  power_cells: 30,
  reactive_armor: 20,
  orbital_shells: 10
});

giveAll(amount) — wszystkiego po równo

// Dodaje `amount` KAŻDEGO surowca i commodity w inwentarzu (domyślnie 1000).
KOSMOS.debug.giveAll();          // po 1000 wszystkiego
KOSMOS.debug.giveAll(5000);      // po 5000 wszystkiego

giveResearch(amount) — punkty badawcze

KOSMOS.debug.giveResearch();       // +10 000 research (domyślnie)
KOSMOS.debug.giveResearch(50000);   // +50 000

giveCredits(amount) — Kredyty handlowe (Kr)

// Waluta handlu cywilnego. Używana do rush build i zakupów awaryjnych.
KOSMOS.debug.giveCredits();        // +10 000 Kr (domyślnie)
KOSMOS.debug.giveCredits(50000);    // +50 000 Kr

Populacja

givePop(amount, strata) — dodaj POPy do aktywnej kolonii

// Strata: laborer (domyślnie), scientist, bureaucrat, specialist
KOSMOS.debug.givePop(5);                   // +5 laborer
KOSMOS.debug.givePop(3, 'scientist');       // +3 naukowców
KOSMOS.debug.givePop(2, 'bureaucrat');      // +2 urzędników

Technologie

unlockTech(...techIds) — natychmiastowe odblokowanie

// Bez argumentów: pakiet militarny (broń, desant, koszary, drony, amunicja orbitalna, logistyka flotowa).
KOSMOS.debug.unlockTech();

// Konkretne techy (dowolna ilość argumentów):
KOSMOS.debug.unlockTech('nuclear_power');
KOSMOS.debug.unlockTech('ion_drives', 'fusion_drives');
KOSMOS.debug.unlockTech('exotic_materials', 'warp_drive', 'quantum_computing');

Wrogowie i walka

spawnTestEnemy() — wroga kolonia do testów desantu

// Tworzy wrogie imperium + kolonię na najbliższym niezamieszkałym ciele.
// Przydatne do testów bitwy orbitalnej, dominacji orbitalnej, desantu i ostrzału.
KOSMOS.debug.spawnTestEnemy();

Przykładowe scenariusze

Szybki start: pełne wyposażenie do testu desantu

// 1. Odblokuj militarne techy
KOSMOS.debug.unlockTech();

// 2. Daj surowce i commodities na budowę statku
KOSMOS.debug.giveAll(2000);

// 3. Daj research do dalszych technologii
KOSMOS.debug.giveResearch(50000);

// 4. Daj POPy do obsługi nowych budynków
KOSMOS.debug.givePop(10);

// 5. Spawn wrogiej kolonii
KOSMOS.debug.spawnTestEnemy();

Tryb "sandbox" — wszystko bez limitów

KOSMOS.debug.unlockTech(
  'point_defense', 'ground_warfare', 'military_logistics',
  'drone_warfare', 'tech_munitions', 'fleet_logistics',
  'nuclear_power', 'ion_drives', 'fusion_drives',
  'quantum_computing', 'exotic_materials'
);
KOSMOS.debug.giveAll(100000);
KOSMOS.debug.giveResearch(1000000);
KOSMOS.debug.giveCredits(1000000);
KOSMOS.debug.givePop(50);
Uwaga: cheaty nadpisują stan zapisu. Po ich użyciu save będzie zawierał dodane zasoby. Nie cofają się automatycznie. Jeśli chcesz wrócić do czystego stanu — załaduj save z przed użycia cheatu lub zacznij nową grę.
